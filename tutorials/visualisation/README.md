# Tutorial to visualise SVs and CNVs from SAVANA


## Step 1: Download small example BAMs

We will perform the tutorial on a small test tumour and matched normal BAM which contains sequencing data from the COLO829 cell line sequenced with ONT, downsampled to 10x sequencing coverage and subset to chromosomes 3, 7, and 10.

```
wget ftp://ftp.ebi.ac.uk/pub/databases/icortes-public/SAVANA_tutorial/*.bam*
```
